### How to use?
```bash
python3 encrypt/decrypt.py –m [method] –i [input] –k [key]
```
-m : Encryption/decryption method.

-i : The text you want to encrypt/decrypt.

-k : The key used for ecryption/decryption.

|Method|Key|
|-|-|
|caesar| shift(int)|
|playfair| key(str)|
|vernam| key(str)|
|railfence| number of row(int)|
|row| order(int)|
